# HR Analytics: Employee Attrition & Workforce Insights

## Project Overview
This project focuses on the **HR Analytics** domain, using data-driven insights to understand why employees leave an organization. By analyzing workforce demographics, satisfaction scores, and salary distributions, this project provides a clear view of attrition patterns to support proactive retention strategies.

## Tech Stack
- **Data Visualization & BI:** Power BI Desktop
- **Data Modeling:** Star Schema (connecting employee performance to satisfaction metrics)
- **DAX (Data Analysis Expressions):** Custom measures for Attrition Rate, Average Salary, and Satisfaction scoring.

## Key Dashboard Insights
The dashboard is designed for HR decision-makers to identify "high-risk" areas:
- **Attrition Overview:** Real-time tracking of the Attrition Rate, active workforce counts, and average tenure.
- **Salary & Attrition Correlation:** Visualizes how salary brackets impact employee retention, highlighting which income groups are most susceptible to turnover.
- **Departmental Turnover:** A comparative analysis showing which departments are experiencing the highest attrition, enabling focused intervention.
- **Satisfaction Analysis:** An interactive matrix visual providing a granular view of job satisfaction, environment satisfaction, and work-life balance scores by job role.
- **Dynamic Filtering:** Slicers for Gender, Department, and Age Group to allow for deep-dive exploratory analysis.

## Data Cleaning & Preparation
To ensure data accuracy, the following steps were performed in Power BI's Power Query:
- **Data Type Casting:** Standardized numerical and categorical types (e.g., converting IDs to text and salary figures to currency).
- **Handling Inconsistencies:** Removed invalid entries and filtered out incomplete records to ensure statistical validity.
- **Feature Engineering:** Calculated the 'Attrition Rate' and grouped data into meaningful segments (e.g., Age Groups and Salary Labs) for better visualization.

## Impact
- Developed a **proactive monitoring tool** that shifts HR focus from reactive hiring to retention.
- Provided actionable insights into which **Job Roles and Departments** require immediate management attention regarding work-life balance and salary benchmarking.
